Record Types

Which record type should you use?

CNAME

Website / GitHub Pages

Point your subdomain to another hostname like username.github.io or myapp.vercel.app

A

Own Server (IPv4)

Point your subdomain to an IPv4 address like 1.2.3.4. Use this if you have a VPS or dedicated server.

AAAA

Own Server (IPv6)

Point your subdomain to an IPv6 address. Works the same as A record but for IPv6.

TXT

Verification / SPF

Used for domain verification (Google, GitHub Pages) or email SPF records.

MX

Email Routing

Route emails sent to your subdomain to a mail server.

NS

Custom Nameserver

Delegate your subdomain to a different DNS provider or nameserver.
